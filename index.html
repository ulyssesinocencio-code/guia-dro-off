Guia do Processo de Drop-off Full

Um guia interativo para envios via indústria.

Bem-vindo ao Guia de Drop-off

Esta aplicação foi criada para simplificar o entendimento do processo de envio de produtos para o nosso centro de distribuição (CD) diretamente pela indústria. O objetivo é fornecer um passo a passo claro e conciso, garantindo que todas as etapas e pré-requisitos sejam cumpridos corretamente para uma operação ágil e sem falhas.

Use o menu à esquerda para navegar entre a visão geral, a lista de pré-requisitos essenciais e o fluxo interativo do processo. Cada seção foi desenhada para focar nas informações mais importantes para você, seller.

Checklist de Pré-requisitos

Antes de iniciar o agendamento do drop-off, é fundamental garantir que todos os pré-requisitos abaixo sejam atendidos. Esta lista serve como um guia de verificação para assegurar que tanto o seller quanto a indústria estejam preparados para o envio, evitando atrasos e problemas na recepção dos produtos.

1

Informar ao time do Full

É obrigatório notificar a equipe do Full que o envio será realizado via indústria.

2

Cadastro de EAN

Todos os itens enviados devem ter o código EAN devidamente cadastrado na Central do Vendedor e no SBS.

3

EAN Visível para Bipagem

O código EAN deve estar impresso e visível na embalagem de cada produto unitário para permitir a leitura no CD.

4

Envio das Etiquetas Master

O seller precisa gerar as etiquetas da caixa master no sistema e enviá-las para a indústria.

5

Etiquetagem da Caixa Master

A caixa master, contendo os itens unitários, deve ser enviada com a etiqueta da Shopee colada.

6

Cadastro do Veículo de Entrega

A indústria é responsável por cadastrar o veículo que fará a entrega no aplicativo do CD.

Processo Passo a Passo

Esta seção detalha o fluxo de trabalho completo para o drop-off, desde o agendamento inicial pelo seller até a entrega final pela indústria. O processo é dividido entre as responsabilidades do **Seller** e da **Indústria**. Clique em cada etapa do fluxograma para ver os detalhes correspondentes ao lado.
